# ranged_int

> ranged_int(min: int = 0, max: int = 100) -> int

Generate a random integer within a range.

**Arguments**

| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `min` | int | 0 | Lower bound (inclusive). Defaults to 0. |
| `max` | int | 100 | Upper bound (inclusive). Defaults to 100. |

**Returns** `int`
