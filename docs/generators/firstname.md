# firstname

> firstname(gender: Literal['male', 'female', 'nonbinary', 'all'] = 'all') -> str

Generate a random first name.

**Arguments**

| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `gender` | Literal['male', 'female', 'nonbinary', 'all'] | 'all' | Filter names by gender. Use 'all' for any gender. |

**Returns** `str`
