# ranged_float

> ranged_float(min: float = 0.0, max: float = 100.0) -> float

Generate a random float within a range.

**Arguments**

| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `min` | float | 0.0 | Lower bound (inclusive). Defaults to 0.0. |
| `max` | float | 100.0 | Upper bound (inclusive). Defaults to 100.0. |

**Returns** `float`
